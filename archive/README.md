# Public-key campaign archive / 公共 Key 活动归档

Public-key campaigns published from 2024 through 2025 have ended. Retired key
values are intentionally omitted from the current documentation to prevent
confusion, automated reuse, and accidental dependence on revoked credentials.

2024 至 2025 年发布的公共 Key 活动均已结束。当前文档不再展示已撤销的 Key
内容，以免造成误解、自动化重复尝试或对失效凭证产生依赖。

For announcement dates and campaign history, see
[GitHub Releases](https://github.com/sunsky89757/FreeGPTapi/releases).

活动发布日期和历史公告请查看
[GitHub Releases](https://github.com/sunsky89757/FreeGPTapi/releases)。

The main [English README](../README.md) and
[中文 README](../README.zh-CN.md) are the only authoritative sources for the
current public-key campaign.

当前公共 Key 活动仅以主 [English README](../README.md) 和
[中文 README](../README.zh-CN.md) 为准。
